# Input

```javascript
import Input from '@mfrm/mfcl/Input'
```

## Description

Inputs accept user entry data. Inputs use labels and sometimes icons to communicate the input field that will accept data when a user submits them.

## Properties

<Props of={Input} />

## Basic Usage

Generic attributes for input elements can be applied.

<Input label="Default" name="name" />

## Size

You can choose between small, medium and large input field sizes, the default being large.

<Input size="sm" label="sm" />
<br />
<Input size="md" label="md" />
<br />
<Input size="lg" label="lg" />

## Required or Disabled

Make input required or disabled.
Disabled attribute cascades to inline button.

<Input type="text" required label="required" />
<br />
<Input type="text" disabled label="disabled" />

## Footnote

Add a small description below the input field.

<Input inputMessage="This is an info message" label="info" />
<br />
<Input error inputMessage="This is an error message" label="error" />
<br />
<Input disabled error inputMessage="This is an disabled error message" label="disabled error" />

## Right Side

   rightSide={

       Log Out
     </Button>
   }
 />
