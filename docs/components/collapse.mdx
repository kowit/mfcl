# Collapse

Collapse creates a smooth transition to hide and show content

## Install

React transition group is needed for the collapse

```bash
yarn add react-transition-group
```

```js
import { Collapse } from '@mfrm/mfcl'
```

## Controlled Collapse

<Component initialState={{ isOpen: false }}>
  {({ setState, state }) => {
    return (
      <>
        <Button onClick={() => setState({ isOpen: !state.isOpen })}>Click Me</Button>
        <Collapse isOpen={state.isOpen}>{mattressIpsum()}</Collapse>
      </>
    )
  }}
</Component>
