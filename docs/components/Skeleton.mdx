# Skeleton

```js
import { Skeleton } from '@mfrm/mfcl'
```

## Description

The Skeleton component allows you to a build out placeholder Skeleton layouts.

## Basic usage

<Skeleton width="25%" />
<Skeleton width="50%" />
<Skeleton width="75%" />
<Skeleton />

## Type

<Skeleton type="text" />
<Skeleton type="image" />
<Skeleton type="button" />
<Skeleton type="button-secondary" />

## Width/Height Units

  <Skeleton type="text" width={25} />
  <Skeleton type="text" width="25" />
  <Skeleton type="text" width="25px" />
  <Skeleton type="text" width="25%" />
  <Skeleton type="text" width="10em" />
  <br />
  <div style={{ height: '200px' }}>
    <Skeleton type="text" height={10} />
    <Skeleton type="text" height="10" />
    <Skeleton type="text" height="10px" />
    <Skeleton type="text" height="10%" />
    <Skeleton type="text" height="5em" />
  </div>
